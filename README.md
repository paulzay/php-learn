# PHP Learn

A structured repository to learn PHP step-by-step.

## Learning Paths

- [Beginner](./beginner/README.md)
- [Mid](./mid/README.md)
- [Advanced](./advanced/README.md)

## How to Use This Repo

1. Start with the beginner track.
2. Move to the mid track after you are comfortable with basics.
3. Finish with advanced topics and architecture.
4. Build one mini-project per track to practice what you learned.
